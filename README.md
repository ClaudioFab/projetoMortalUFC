Simulador de Lutas UFC (Java)

Descrição:

Simulador de lutas estilo UFC desenvolvido em Java utilizando Programação Orientada a Objetos. O sistema cria lutadores, valida combates e simula lutas em rounds com resultados aleatórios.

Funcionalidades:

* Criação de lutadores com atributos e histórico
* Definição automática de categoria por peso
* Validação de lutas (mesma categoria e lutadores diferentes)
* Seleção aleatória de lutadores
* Simulação de 3 rounds com resultados aleatórios
* Atualização de vitórias, derrotas e empates
* Exibição de placar final
* Interação com JOptionPane (interface gráfica simples)

Estrutura:

* Main_UFC

Controla execução do programa, sorteio de lutadores e fluxo da luta.

* Lutador

Gerencia atributos, categoria automática e histórico de lutas.

* Luta

Responsável por validar, executar rounds e definir o vencedor.


Como Executar:

Compile:

javac projeto_ufc/*.java

Execute:

java projeto_ufc.Main_UFC

Tecnologias
* Java
* POO
* JOptionPane
* Random

Objetivo:

Praticar conceitos de Programação Orientada a Objetos como encapsulamento, classes, métodos e interação entre objetos.
